# Pokedex DIO

Projeto simples de Pokedex desenvolvido como atividade da DIO.

## Descrição

Trata-se de uma aplicação web estática que consome a API pública do PokéAPI para listar pokémons, pesquisar por nome e carregar mais resultados. Os cards foram estilizados com um visual moderno, com topo vermelho e base branca, fonte `Press Start 2P` e imagens centralizadas.

## Funcionalidades

- Carrega uma lista inicial de pokémons
- Pesquisa pokémon por nome
- Carrega mais pokémons ao clicar no botão
- Cards com visual responsivo e imagem centralizada
- Layout em duas cores (vermelho em cima, branco embaixo)

## Tecnologias

- HTML
- CSS
- JavaScript
- PokéAPI

## Como usar

1. Abra o arquivo `index.html` no navegador
2. Use o campo de busca para pesquisar um pokémon pelo nome
3. Clique em `Carregar mais` para carregar mais pokémons

## Estrutura de arquivos

- `index.html` — marcação da página
- `style.css` — estilos visuais dos cards e layout
- `script.js` — lógica de busca e carregamento dos pokémons

## Observações

- Projeto está feito para ser executado localmente sem servidor.
- Requer conexão com a internet para acessar a PokéAPI.
